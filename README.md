## 🔧 Установка и запуск
Скачать проект
Распаковать
Запустить main.exe

P.S: Не перемещать файлы и папки


### Требования к компиляции
- C++ компилятор (g++, MSVC, clang)
- raylib (Можно загрузить прямо в папку)
- При компиляции не использовать линковку

### Компиляция
```bash
g++ main.cpp -o main.exe -lraylib -lopengl32 -lgdi32 -lwinmm
```
